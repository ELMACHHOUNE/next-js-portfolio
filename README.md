## Getting Started
Prerequisites:
- Node.js 18+ and npm 9+

Install:
- npm install

Development:
- npm run dev
- Open http://localhost:3000

Build & Run:
- npm run build
- npm start

Lint (if configured):
- npm run lint

## Suggested Structure
- src/
  - components/
  - pages/ or app/
  - styles/
- public/

## Deployment (Vercel)
- Push to GitHub
- Import repo in Vercel
- Deploy with defaults for Next.js

## SEO Tips
- Update metadata (title, description, Open Graph)
- Add favicon in /public
